# QUICK START GUIDE

## Step 1: Setup Environment

### Option A: Using venv (Windows)
```bash
# Create virtual environment
python -m venv venv

# Activate it
venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt
```

### Option B: Using conda
```bash
# Create environment
conda create --name patient_risk python=3.11

# Activate it
conda activate patient_risk

# Install dependencies
pip install -r requirements.txt
```

## Step 2: Generate Sample Data (for testing)

```bash
python generate_sample_data.py
```

This creates `data/raw/patient_data.csv` with 1000 sample patient records.

## Step 3: Run the Pipeline

```bash
python main.py
```

The pipeline will:
- Load and validate patient data
- Generate EDA visualizations
- Clean and preprocess data
- Engineer clinical features
- Train multiple ML models
- Evaluate and select best model
- Generate reports and risk scores

All outputs go to:
- `reports/` - Visualizations and evaluation reports
- `models/` - Trained model and configuration
- `data/processed/` - Cleaned dataset

## Step 4: Run the API Server

```bash
# Option 1: Direct
cd api
python main.py

# Option 2: Using uvicorn
uvicorn api.main:app --reload

# Option 3: Using Docker
docker-compose up
```

API runs on http://localhost:8000

## Testing with Your Own Data

1. Prepare CSV with these columns:
   - age, bmi, gender (demographics)
   - systolic_bp, diastolic_bp, heart_rate, temperature, respiratory_rate (vitals)
   - glucose, creatinine, hemoglobin, wbc_count, platelets (labs)
   - diabetes, hypertension, heart_disease, copd, chronic_kidney_disease (comorbidities)
   - medication_count, antibiotic_use, corticosteroid_use (medications)
   - readmission_risk (target - 0 or 1)

2. Place in `data/raw/your_data.csv`

3. Update file path in `main.py` or load directly:
   ```python
   from src.data_loader import DataLoader
   loader = DataLoader('data/raw/your_data.csv')
   data = loader.load_data()
   ```

## Troubleshooting

**Model not loaded in API?**
- Run the pipeline first: `python main.py`
- Check that `models/best_model.joblib` exists

**Missing features error?**
- Ensure your data has all required columns
- Check `models/model_config.json` for expected features

**Docker issues?**
- Install Docker: https://www.docker.com/products/docker-desktop
- Build image: `docker-compose build`
- Run: `docker-compose up`
